This repository contains an Arduino class that provides the MCU with a web interface used to both set the MQTT broker IP address as well as do OTA firmware updates. This is part of a series of classes made to support a standard set of APIs for experimental robot platforms.\hypertarget{index_autotoc_md9}{}\doxysubsection{Dependencies}\label{index_autotoc_md9}
This class depends on the following libraries being present on your system\+:


\begin{DoxyItemize}
\item \href{https://github.com/espressif/arduino-esp32}{\texttt{ Arduino.\+h}} This is the Arduino core library that comes bundled with Platform\+IO.
\item Update.\+h. This comes with the Arduino core library.
\item Wi\+Fi\+Client.\+h. Required for hosting a web server. This comes with the Arduino core library.
\item Web\+Server.\+h. Required for hosting a web server. This comes with the Arduino core library.
\item ESPm\+DNS.\+h. Required for hosting a web server. This comes with the Arduino core library.
\item \href{https://github.com/marian-craciunescu/ESP32Ping}{\texttt{ ESP32\+Ping.\+h}} Used to verify IP addresses, specificlly the MQTT broker IP address.
\item \href{https://github.com/theAgingApprentice/aaNetwork}{\texttt{ aa\+Network.\+h}} Required for Wifi connection.
\end{DoxyItemize}\hypertarget{index_autotoc_md10}{}\doxysubsection{Author}\label{index_autotoc_md10}
